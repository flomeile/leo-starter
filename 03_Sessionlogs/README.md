# 03_Sessionlogs

Enthält die Zusammenfassungen einzelner KI-Sessions, erzeugt durch den Wrap-Up-Skill. Thematisch in Unterordner sortiert (die legst du an, sobald du Themen hast).

## Dateinamen
Format: `YYYY-MM-DD-HHmm-slug.md`

## Regel für das LLM
Neu abgelegte Sessionlogs sind ab Ablage sofort über Index und Filesystem auffindbar. Greife aktiv darauf zurück, wenn eine Frage früheres erarbeitetes Wissen betrifft. Sessionlogs sind datierte Zustandsaufnahmen (Archiv); bei Widerspruch schlägt die kuratierte Wissensdatei den Log.
