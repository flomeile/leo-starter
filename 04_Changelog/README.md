# 04_Changelog

Enthält `Changelog.md`, das Protokoll aller Änderungen an den Dateien im Basiskontext (`01_Basiskontext`).

## Regel für das LLM
Jede Änderung an einer Basiskontext-Datei wird hier mit Datum, betroffener Datei und Kurzbeschreibung eingetragen. So bleibt nachvollziehbar, was sich wann geändert hat.
