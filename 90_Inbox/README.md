# 90_Inbox

Rohablage. Hier landet, was noch keinem Thema zugeordnet ist: importierte Deep-Research-Berichte, lose Notizen, Rohmaterial.

## Regel für das LLM
Der Standardweg aus der Inbox ins Repo: Inhalt verlustfrei in eine Wissensnotiz im passenden Themenordner konsolidieren, Quelle zitieren, Rohquelle danach (git-gesichert) löschen. Der System-Health-Check (Schritt Inbox) macht nur Ablage-Hygiene: unverändert passende Dateien mit Bestätigung verschieben, Verarbeitungsfälle als Vorschlag melden. Der Index erfasst auch Inbox-Dateien.
